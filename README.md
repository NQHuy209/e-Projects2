# Đề tài: Phần mềm quản lý rạp chiếu phim

## Các chức năng chính

### Chức năng quản lý:
1. Quản lý phim
2. Quản lý suất chiếu
3. Quản lý người dùng
4. Quản lý hóa đơn

### Chức năng bán hàng:
1. Bán vé xem phim

### Chức năng thống kê:
1. Thống kê doanh thu phim


### Các chức năng khác:
1. Đăng nhập, đăng xuất
